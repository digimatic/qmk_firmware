# The digimatic keymap for Lily58 KB (splitkb.com)

My switches are Kailh Speed Bronze: Clicky | 1.1mm operating | 3.5mm total | 50g Actuation | 65g Bottom Out



Build hex:
```sh
make lily58:digimatic
```

Build and flash:
```sh
make lily58:digimatic:dfu
```
